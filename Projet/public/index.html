CarViz - A data vizualisation project

L'idée de ce projet était de faire de l'exploration design au niveau de la visualisation de données. Nous sommes partis pour cette première illustration sur un graphique en barre et y avons apondu un design de voiture qui est généré à partir de path uniques. L'effet de "vitesse" sont illustrés à travers ces deux bandes colorés qui pourrait systhéthiser les voitures qui driftent. Cet effet a été réalisé en superposant un rectangle sur l'autre.

Attention le temps affiché est un calcul fictif pour calculer le 0 à 100km/h. L'idée était de pouvoir récupérer les données de poids, puissance pour en faire un calcul et y afficher le resultat.

Force directed graph

Le challenge ici était de faire une visualisation de données intéractive, nous avons utilisé la librairie AMCHARTS pour réaliser ce graph de type "FORCE DIRECTED".

Nous avons réussi à mettre en place une méthode récursive qui parcourt le tableau de données tant que le constructeur a des éléments enfants afin d'y rajouter des coulurs personnalisées en fonction du niveau de hiérarchie. Par des soucis de temps et de priorités, nous n'avons pas intégré les données depuis un fichier JSON externe mais depuis une variable JS directement.
